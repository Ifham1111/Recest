First these are the type of writing funcion what are the function types in react js 1. Class Component 2. Functional Component what is a class component? A class component is a javascript class that extends the React.Component class. It has access to all the functions of the React.Component class such as render(), componentDidMount(), etc. It also has access to the state object and lifecycle methods. It is also called a stateful component. what is a functional component? A functional component is a javascript function that returns a React element. It is also called a stateless component as it does not have access to the state object and lifecycle methods. It is also called a dumb component as it does not have any internal state of its own. It is also called a presentational component as it is mainly used for presenting the UI. 2. What is the difference between state and props? State is mutable whereas props are immutable. State is used to update the component whereas props are used to render the component. State is internal and controlled by the component itself whereas props are external and controlled by whatever renders the component.
